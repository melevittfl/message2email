Welcome to Message2Email.

This project is a small Flask utility to process incoming SMS messages. Messages can be split into multiple parts and the bulk of the interesting bit is processing those multipart messages.

Note: It currently only handles multi-part messages from Nexmo, not twilio.

It uses memcached to store each part of a message and then e-mails them using Postmark.
